# pure-web
使用stylish去除网络上的广告和各种垃圾信息，同时尽量不改变网站原来的结构

[stylish地址](https://userstyles.org/styles/156465/pure-web)

## 目前支持的站点
1. 微博的广告和冗余信息
2. 百度的广告: 百度搜索页、百度知道、百度翻译、百度百科
3. 知乎的广告
4. iteye的广告
5. cnblogs的广告
6. csdn的广告
7. reddit的广告
8. 网易博客的广告
9. 新浪博客
10. 开源中国
11. v2ex
12. runoob.com
